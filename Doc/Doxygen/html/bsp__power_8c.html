ERROR_CODE_E bsp_pwr_set_scaling ( const POWER_SCALING_E  pwr_scaling)

Sets the power scaling.

Sets the power scaling based on the values passed in paramter. The scaling conforms to the values defined by the reference manual. If the value does not conform, the function has no effect and returns an error code.

Parameters
pwr_scalingThe scaling factor configuration.
Returns
NO_ERROR is returned in case of success. Otherwise an error code is returned. Please refer to the list of the standard error codes.
